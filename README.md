# Luxury Estates Platform

A premium real estate marketplace for browsing, filtering, and comparing luxury properties across the globe, featuring interactive maps, mortgage calculations, and agent chat.

## 🚀 Features

* **Interactive Maps:** Visualizes property locations using Leaflet (OpenStreetMap).
* **Advanced Filtering:** Search properties by location, name, or description.
* **Sorting:** Sort listings by price (high/low) or alphabetical order.
* **Favorites:** Save properties to local storage to view them in your profile.
* **Property Comparison:** Compare multiple luxury properties side-by-side.
* **Mortgage Calculator:** Estimate monthly payments on the property details page.
* **Agent Live Chat:** Simulated chat interface to interact with agents.

## 🛠️ Technology Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Maps:** React Leaflet
* **Routing:** React Router DOM
